## How to Use These Materials

### 1. Review the Presentation

Start with `MCP-Context-Forge-Presentation.md` to understand the slide structure and content. Each slide is presented in ASCII box format that can be directly converted to PowerPoint slides.

### 2. Practice with Speaker Script

Use `Speaker-Script.md` while practicing. It includes:
- Exact timing for each slide
- Complete spoken text for each section
- Transition phrases between slides
- Anticipated Q&A with suggested answers

### 3. Prepare the Demo

Follow `Demo-Script.md` to set up your demo environment:
- Start MCP Context Forge locally
- Register sample tools
- Prepare browser tabs and terminal

### 4. Create PowerPoint Slides

Convert the ASCII slides to PowerPoint:
1. Create slides matching the structure in `MCP-Context-Forge-Presentation.md`
2. Use IBM branding guidelines for colors and fonts
3. Add the IBM logo to title and closing slides
4. Consider adding screenshots from the Admin UI

## Key Messages

1. **MCP provides the standard** - A consistent, open protocol for AI-tool interaction
2. **MCP Context Forge provides the control** - Centralized governance, security, and federation
3. **Together they enable scale** - Production-ready infrastructure for agentic AI
